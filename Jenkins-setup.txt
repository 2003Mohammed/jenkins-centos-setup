## Step 1: Add Jenkins Repository

```bash
sudo wget -O /etc/yum.repos.d/jenkins.repo \

```

## Step 2: Install Java (Required)

Jenkins requires Java. Here we use **Java 21**:

```bash
sudo yum install -y fontconfig java-21-openjdk
```

Verify:

```bash
java -version
```

---

## Step 3: Install Jenkins

```bash
sudo yum install -y jenkins
```

Reload systemd:

```bash
sudo systemctl daemon-reload
```

---

## Step 4: Change Jenkins Port (8080 â†’ 8090)

By default, Jenkins runs on **port 8080**. We change it to **8090**.

### Option A (Common): Edit Jenkins Environment File using insert mode by clicking lowercase i

```bash
sudo vi /etc/sysconfig/jenkins
```

Find:

```bash
JENKINS_PORT="8080"
```

Change to:

```bash
JENKINS_PORT="8090"
```

Save and exit using esc and :wq .

### Option B (If environment is defined in service file)

```bash
sudo vi /usr/lib/systemd/system/jenkins.service
```

Look for:

```bash
Environment="JENKINS_PORT=8080"
```

Change to:

```bash
Environment="JENKINS_PORT=8090"
```

Reload systemd again:

```bash
sudo systemctl daemon-reload
```

---

## Step 5: Start & Enable Jenkins

Enable Jenkins at boot:

```bash
sudo systemctl enable jenkins
```

Start Jenkins:

```bash
sudo systemctl start jenkins
```

Check status:

```bash
sudo systemctl status jenkins
```
